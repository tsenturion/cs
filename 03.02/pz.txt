Напишите код, который создает новый ключ в разделе HKEY_CURRENT_USER\Software. Назовите ключ MyRegistryTest.
В этом ключе создайте следующие значения:
Значение по умолчанию (пустая строка) с текстом "Это тестовый ключ".
Строковое значение с именем TestString и значением "Hello, World!".
Целочисленное значение с именем TestInt и значением 42.

Напишите код, который считывает и выводит на экран значения, созданные на предыдущем шаге
Если значение не существует, выводите сообщение о том, что оно отсутствует

Измените значение TestString на "Измененное значение" и обновите значение TestInt до 100

Напишите код для удаления значения TestString из созданного ключа

После завершения всех операций удалите созданный ключ MyRegistryTest из реестра

Теория:

Microsoft.Win32

Registry: Класс для доступа к корневым разделам реестра и выполнения операций с ними
RegistryKey: Класс, представляющий ключ реестра

Класс Registry
Registry.CurrentUser: Свойство, возвращающее корневой ключ HKEY_CURRENT_USER.
Registry.SetValue(string keyName, string valueName, object value): Метод для записи значения в указанный ключ реестра.
Registry.GetValue(string keyName, string valueName, object defaultValue): Метод для получения значения из указанного ключа. Возвращает значение по умолчанию, если значение не существует.

Класс RegistryKey
OpenSubKey(string name): Метод для открытия подраздела реестра.
GetValueNames(): Метод для получения массива имен всех значений в текущем ключе.
GetValue(string name): Метод для получения значения по имени из текущего ключа.
GetValueKind(string name): Метод для определения типа значения по имени (например, RegistryValueKind.String, RegistryValueKind.DWord, и т.д.).
DeleteValue(string name): Метод для удаления значения из текущего ключа.
DeleteSubKey(string subkey): Метод для удаления подраздела из текущего ключа.
